# Driving Theory Quiz App

A TanStack Start (React + TypeScript) full-stack app with Supabase for auth and data. Built originally on Lovable.dev.

## Stack

- **Framework**: TanStack Start (SSR React with TanStack Router)
- **Styling**: Tailwind CSS v4 + Radix UI / shadcn components
- **Auth & DB**: Supabase (`@supabase/supabase-js`)
- **Build**: Vite 8 via `@lovable.dev/vite-tanstack-config`
- **AI**: Vercel AI SDK (`ai`, `@ai-sdk/openai-compatible`)

## Running

```bash
npm run dev
```

Serves on port 5000.

## Required Secrets

- `VITE_SUPABASE_URL` — Supabase project URL
- `VITE_SUPABASE_PUBLISHABLE_KEY` — Supabase anon/publishable key
